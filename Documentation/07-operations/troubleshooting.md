# Troubleshooting

Find solutions to common issues encountered with ToolboxAI-Solutions.

## Common Problems

### 1. API returns 401 Unauthorized

- **Solution:** Check your API key or token. Ensure it is valid and not expired.

### 2. Data not processing

- **Solution:** Verify data format and required fields. Check job status in dashboard.

### 3. Slow response times

- **Solution:** Review system load and rate limits. Contact admin if persistent.

### 4. Cannot log in

- **Solution:** Reset password or contact support for account recovery.
