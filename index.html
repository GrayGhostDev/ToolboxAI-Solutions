    üéØ Quick Start (5 Minutes)¶

    Prerequisites¶

    • Docker 25.x with Docker Compose v2
    • Git
    • 4GB+ RAM available for Docker

    One-Command Setup¶

    # Clone repository
git clone https://github.com/ToolBoxAI-Solutions/toolboxai.git
cd toolboxai

# Start all services (Docker recommended)
docker compose -f infrastructure/docker/compose/docker-compose.yml \
               -f infrastructure/docker/compose/docker-compose.dev.yml up -d

# Access the platform
# Dashboard: http://localhost:5179
# Backend API: http://localhost:8009
# API Docs: http://localhost:8009/docs

    External Drive Development

    If developing on an external drive (e.g., /Volumes/ on macOS):

    • Recommended: Use Docker for all development
    • Issue: Native binaries (esbuild, Rollup) fail with system error -88
    • NPM Workaround: npm install --no-bin-links --legacy-peer-deps
    • Vite Config: Use vite.config.js instead of .ts
