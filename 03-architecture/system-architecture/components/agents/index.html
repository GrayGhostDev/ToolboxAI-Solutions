    Agent Design Documents¶

    LessonAnalysisAgent¶

    • Responsibility: Parse lesson plans, map to standards
    • Input: Lesson plan (text, PDF, DOC)
    • Output: Structured lesson objects
    • Prompts: Curriculum mapping, error handling

    EnvironmentAgent¶

    • Responsibility: Generate Roblox environment specs
    • Input: Lesson objects
    • Output: Environment blueprints
    • Prompts: Scene layout, NPC logic

    ObjectAgent¶

    • Responsibility: Define game objects
    • Input: Environment specs
    • Output: Object definitions
    • Prompts: Object placement, attributes

    ScriptAgent¶

    • Responsibility: Generate scripts for Roblox
    • Input: Object definitions
    • Output: Lua scripts
    • Prompts: Script templates, error handling

    ValidationAgent¶

    • Responsibility: Validate environment and scripts
    • Input: Environment, scripts
    • Output: Validation report
    • Prompts: Error detection, compliance checks
