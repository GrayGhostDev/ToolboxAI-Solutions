# GitHub Projects Integration

This repository includes comprehensive GitHub Projects integration for project management, sprint planning, and issue tracking.

## Quick Start

```bash
# Check prerequisites
./scripts/github-projects/quick-start.sh

# Create projects
./scripts/github-projects/setup-projects.sh

# Import issues
gh project list --owner GrayGhostDev
./scripts/github-projects/create-project-items.sh <project-number>
```
