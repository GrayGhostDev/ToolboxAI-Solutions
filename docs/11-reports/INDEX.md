## üõ†Ô∏è Quick Reference

### Common Commands

```bash
# Development
pnpm run dev                    # Start full stack
pnpm run dashboard:dev          # Frontend only
cd apps/backend && uvicorn main:app --reload  # Backend only

# Testing
pytest                          # Backend tests
pnpm run dashboard:test         # Frontend tests
pnpm run test:e2e              # E2E tests

# Database
alembic upgrade head            # Apply migrations
alembic revision -m "msg"       # Create migration
python scripts/database/seed_data.py  # Seed data

# Deployment
git push origin main            # Auto-deploy
pnpm run deploy:all            # Manual deploy

# Docker
pnpm run docker:up             # Start services
pnpm run docker:down           # Stop services
pnpm run docker:logs           # View logs
```

### Important Ports

| Service | Port | URL |
|---------|------|-----|
| Backend API | 8009 | http://localhost:8009 |
| Dashboard | 5179 | http://localhost:5179 |
| API Docs | 8009 | http://localhost:8009/docs |
| PostgreSQL | 5434 | localhost:5434 |
| Redis | 6381 | localhost:6381 |

### Key Files

| File | Purpose |
|------|---------|
| `apps/backend/main.py` | Backend entry point |
| `apps/dashboard/src/App.tsx` | Frontend entry point |
| `render.yaml` | Render deployment config |
| `vercel.json` | Vercel deployment config |
| `docker-compose.yml` | Docker services |
| `.env` | Environment variables |
