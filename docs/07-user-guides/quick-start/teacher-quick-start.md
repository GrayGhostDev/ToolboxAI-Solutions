# 5-Minute Teacher Quick Start Guide

Get up and running with ToolBoxAI in just 5 minutes! This guide will have you creating your first interactive lesson in no time.

## Prerequisites âœ…

- [ ] Educator account created by your administrator
- [ ] Login credentials received
- [ ] Roblox account (personal or school-provided)
- [ ] Lesson plan ready (PDF, Word, or text)

## Step 1: Login and Setup (1 minute)

1. **Access the Platform**
   - Go to your school's ToolBoxAI portal
   - Login with your credentials
   - Complete the 1-minute welcome tour

2. **Profile Setup**
   - Upload a profile picture
   - Add your subjects and grade levels
   - Set notification preferences

## Step 2: Create Your First Class (1 minute)

1. **Add a New Class**
   - Click "Classes" â†’ "Add New Class"
   - Enter class name: "Science 7A" (example)
   - Set subject: Science, Grade: 7
   - Click "Create Class"

2. **Get Your Class Code**
   - Copy the generated class code
   - Share with students for enrollment

## Step 3: Generate Your First Interactive Lesson (2 minutes)

1. **Start Lesson Creation**
   - Click "Create New Lesson" from dashboard
   - Choose "AI-Powered Generation"

2. **Upload Your Content**
   - Upload your lesson plan (PDF/Word)
   - OR paste lesson text directly
   - Add title: "The Solar System"
   - Select grade level and subject

3. **Choose Environment**
   - Select "Space Station" theme
   - Enable "Exploration Mode"
   - Add "Interactive Quiz Checkpoints"
   - Click "Generate"

4. **Wait for Magic** âš¡
   - AI creates your 3D world (30-60 seconds)
   - Review generated environment
   - Make quick customizations if needed
   - Click "Publish to Class"

## Step 4: Invite Students (1 minute)

1. **Share Class Code**
   - Send class code to students
   - OR use bulk email feature
   - Students join using the code

2. **Launch Your Lesson**
   - Students access via dashboard
   - Roblox opens automatically
   - Monitor progress in real-time
