## ğŸš€ Automatic Deployment

Documentation is automatically built and deployed when:
- Changes are pushed to `main` branch in the `docs/` directory
- Manual workflow trigger via GitHub Actions

The workflow:
1. Builds the Jekyll site
2. Generates API documentation from OpenAPI specs
3. Creates Swagger UI for interactive API docs
4. Validates links (on PRs)
5. Deploys to GitHub Pages

## ğŸ› Troubleshooting

### Build fails locally

```bash

bundle exec jekyll clean

```

### Links broken after deployment

Check `baseurl` in `_config.yml`:
```yaml
baseurl: "/ToolboxAI-Solutions"
```

### Images not loading

Ensure images use absolute paths:
